2. Explain when a database would be used. Explain when a text file would be used.

A: Text files should only be used if a database cannot be used for whatever reason or in the very simplest cases where there is no need to query or modify data. Databases should be used whenever possible.


3. Describe one difference between SQL and other programming languages.

A: SQL is devlarative, like html or css, meaning that we declare what we want to access or display, as opposed to procedural languages like javascript where we describe how the operation should proceed.

4. In your own words, explain how the pieces of a database system fit together at a high level.

A: The database contains objects called tables that are made up of vertical columns and horizontal rows. Columns define what the data represents and each row makes up a complete conceptual unit. The intersections of these are called cells and they contain diferent pieces of data.

5. Explain the meaning of table, row, column, and value.

A: 
table: a data structure made up of rows and columns.
row: vertical column that defines what the data represents.
column: horizontal row that contains the data for a single conceptual unit.
value: the actual data stored in the cells, can be of any data type.
